A) Quick Start - top

Upload the folder wolfhunter onto a server, and browse to it! For example, if you have a "games" directory on your site, and upload the "wolfhunter" folder there, you'd play it at: http://www.example.com/games/wolfhunter

B) Game Description - top

'Wolf Hunter' The hunter becomes the hunted in this take on 'whack a mole'. Load and reload your shotgun and touch/click the hunters as they emerge from behind the cover. The blue hunter takes one shot and the red hunter takes two shots to kill. They could appear from behind any cover and the games speed will gradually increase to a chaotic crescendo, the aim is to get the maximum score before the game ends. Be careful not to hit the Wolf. The wolves can only take one hit and will rapidly deplete your score.

C) Framework - top

Wolf Hunter uses the PHASER HTML5 Framework

PHASER documentation : http://docs.phaser.io PHASER examples http://examples.phaser.io

Phaser Framework

All Phaser files related to the framework are located within /js

  • phaser.js

Game Files

All files specific to the Game are located within /src

  • Boot.js
    • Prepares initial Preloader images, Global variables
  • Preloader.js
    • Preloads all game assets and starts MainMenu.js on loaded
  • MainMenu.js
    • Presents the Main Menu Screen
  • Game.js
    • Main game

D) Game Library Items - top

Audio

/audio this folder contains all the sound fx's. Audio files have been provided in .ogg and .mp3 to provide compatibility with Desktop, Android(.ogg) and Apple(.mp3).

GAME IMAGES

/images/ folder provides all the imges for the game including: the game,preloader and menu background, orientation indicator, preloader,spritesheets and character artwork

E) Customisable Features - top

The graphics, sound fx's and text can all be customised as desired.

Customising the game is a simple process of creating your custom image or sound and making sure the original name and location are preserved.

Character animations

The character animations apply to the blue,red hunter and the wolf and have been created with Texturepacker (file included).

The blue and red hunter in play animations consists of 7 images: blue = hunter0001.png - hunter0007.png red = hunter20001.png - hunter20007.png

Wolf animation contains 5 images: wolf0001.png - wolf0005.png

All Other Images

The look of the whole game can be customised to suit your requirements

All the images can be customised by replacing the original image with your custom image. The height x width sizes must remain the same.
